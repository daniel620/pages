# Python for Beginners

Start your journey with Python programming.

## Basic Syntax

```python
# Variables
name = "Alice"
age = 30

# Functions
def greet(person):
    return f"Hello, {person}!"

# Classes
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"I'm {self.name}, {self.age} years old."
```

## Data Structures

```python
# Lists
fruits = ["apple", "banana", "orange"]

# Dictionaries
person = {
    "name": "Bob",
    "age": 25,
    "city": "New York"
}

# List comprehension
squares = [x**2 for x in range(10)]
```
